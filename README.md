# d2c-101

A single-page visualization of the eight core unit-economics metrics for direct-to-consumer businesses and how they connect.

## View

Once GitHub Pages is enabled (Settings → Pages → Deploy from branch: `main` / root):

https://cyyeh.github.io/d2c-101/metrics.html

## Source

Metrics, formulas, and operating principles are summarized from Mailchimp's *Ecommerce Playbook*, Chapter 3 — *How to get your unit economics right* (with Mehtab Bhogal of Karta Ventures):

https://mailchimp.com/marketing-glossary/ecommerce-playbook/#Chapter_3%3A_How_to_get_your_unit_economics_right
